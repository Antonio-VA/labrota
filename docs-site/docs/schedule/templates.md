---

Templates let you save a week's rota and apply it to future weeks as a starting point.

## Saving a template

1. Generate or edit a rota you're happy with
2. Click the **⋯** overflow menu in the toolbar
3. Choose **Save as template**
4. Give it a name (e.g. "Summer staffing", "4-person week")

## Applying a template

1. Navigate to the week you want to fill
2. Click **⋯ → Apply template**
3. Select the template from the list
4. The template assignments are applied — you can then edit as needed

Templates copy shift assignments but not leave data, so staff on leave in the target week will appear as conflicts you can resolve manually.

## Copy previous week

A quicker option: click **⋯ → Copy previous week** to copy all assignments from the week immediately before. Useful when the rota changes little from week to week.
