title: Shift Types
---

Shift types define the shifts that staff can be assigned to. They appear as rows in the week view grid.

## Viewing shift types

Go to **Lab → Shift types** to see all configured shifts with their start and end times.

## What the engine uses

When generating a rota, the AI engine assigns staff to shifts based on:

- The shift's start and end time
- Which days of the week the shift is active
- Required coverage ratios (set in Lab config)

## Time display

Times can be shown in **12h** or **24h** format. Change this in **Settings → Lab → Time format**.
