---

LabRota tracks which IVF techniques each staff member is certified to perform. The AI engine uses this to ensure every shift has qualified coverage for each technique.

## Adding a skill

1. Open a staff member's profile (click their name in the Team list)
2. Click the **Skills** tab
3. Click **Add skill** and select the technique
4. Set the level: **Training** or **Certified**

Only **Certified** skills count for coverage purposes. **Training** is informational.

## Skill gap warnings

If a scheduled shift has no certified staff for a required technique, the rota will show a **skill gap** warning (red indicator on that day). Generate again or manually add a qualified staff member to resolve it.

## Techniques (Técnicas)

The available techniques are configured in **Lab → Techniques**. Common examples:

- ICSI
- FIV clásica
- Biopsia embrionaria
- Vitrificación
- Desvitrificación

Your lab administrator can add or rename techniques to match your specific workflows.
